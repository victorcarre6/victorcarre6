## LLM & Productivity Tools

- **[LocalMind](https://github.com/victorcarre6/llm-assistant)** &nbsp; <img src="https://img.shields.io/badge/Open%20Source-brightgreen?logo=opensourceinitiative&logoColor=white" height="18" />
